Detailed Description

Merge and hoist If nodes

  1. Merge consecutive If nodes with the same condition

E.g. Transform

if (x) { f(); }
if (x) { g(); }

into

if (x) {
f();
g();
}
  1. Hoist loop-independent If nodes

E.g. Transform

for i = 0 to n {
if (n < 100) { ... }
}

into

if (n < 100) {
for i = 0 to n { ... }
}

We do not handle else-cases in this pass. Otherwise, the resulting code will be too long. This is different from schedule/separate_tail
