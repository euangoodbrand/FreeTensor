Detailed Description

Lazy full derivative dy/dx_i for all x_i that continuously infuences y

Notes on non-differential cases:

  • If some x_i is missing from a LazyFullDerivative, it means x_i is not continuously infuences y, for example when x_i is an integer. So we can safely ignore all missing items.
  • If error_ is set, it means there is some errors or unsupported cases. It will be re-thrown if calling genGrads.
